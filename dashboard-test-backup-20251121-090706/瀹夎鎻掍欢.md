# VSCode插件安装指南

## 方法1：VSCode内安装（推荐）

### 1. Auto Rename Tag
```
1. 打开VSCode
2. 按 Ctrl+Shift+X 打开扩展商店
3. 搜索 "Auto Rename Tag"
4. 点击 Install
```

### 2. Bracket Pair Colorizer 2
```
搜索 "Bracket Pair Colorizer 2"
点击 Install
```
**注意**：VSCode新版本已内置此功能，可在设置中启用：
```
File → Preferences → Settings
搜索 "bracket pair"
勾选 "Editor > Bracket Pair Colorization: Enabled"
```

### 3. HTMLHint
```
搜索 "HTMLHint"
点击 Install
```

---

## 方法2：命令行安装

打开VSCode，按 Ctrl+` 打开终端，然后运行：

```bash
code --install-extension formulahendry.auto-rename-tag
code --install-extension coenraads.bracket-pair-colorizer-2
code --install-extension mkaufman.htmlhint
```

---

## 验证安装

1. 打开 index.html
2. 点击任意 `<div>` 标签
3. 如果看到：
   - 配对的标签高亮显示 → ✅ 括号配对功能生效
   - 标签有颜色编码 → ✅ Bracket Pair Colorizer生效
   - 波浪线标记错误 → ✅ HTMLHint生效

---

## 启用设置（可选）

在VSCode设置中添加：

```json
{
  "editor.bracketPairColorization.enabled": true,
  "editor.guides.bracketPairs": true,
  "editor.matchBrackets": "always",
  "htmlhint.enable": true
}
```

按 Ctrl+Shift+P → 输入 "Preferences: Open Settings (JSON)" → 粘贴上述内容

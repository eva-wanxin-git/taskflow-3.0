# VSCode Bookmarks for Dashboard Modules

## 使用方法
1. 安装VSCode插件: Bookmarks (alefragnani.Bookmarks)
2. 打开 index.html
3. 按下面的行号，在每个位置按 Ctrl+Alt+K 添加书签
4. 按 Ctrl+Alt+L 查看所有书签

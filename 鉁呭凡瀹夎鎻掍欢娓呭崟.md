## 📋 AI工作流（标准流程）

### 接任务时（自动执行）

```bash
# 1. 备份当前版本
Copy-Item index.html index.html.backup-$(Get-Date -Format 'yyyyMMdd-HHmmss')

# 2. 生成模块索引
python 生成模块索引.py

# 3. 运行诊断脚本
python check_balance.py

# 4. 打开导航页面
Start-Process 模块导航.html
```

### 修改代码时（边改边备份）

```bash
# 修改前备份
Copy-Item file.py file.py.backup-before-fix

# 修改代码
...

# 修改后验证
python check_xxx.py
```

### 完成任务时（交接准备）

```bash
# 创建版本备份
New-Item -ItemType Directory -Path project-v1.x-final

# 复制关键文件
Copy-Item *.html, *.py, *.md

# 重启服务器测试
python -m http.server 8820

# 生成完成报告
echo "✅完成报告.md"
```

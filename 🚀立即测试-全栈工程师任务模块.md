## 🔍 如果看不到任务列表

### 检查1: Console错误
查看是否有红色错误，可能的错误：
- `loadEngineerTasks is not defined` → 函数未定义
- `Failed to fetch` → API未运行
- `Unexpected token` → JavaScript语法错误

### 检查2: 网络请求
F12 → Network标签:
- 查找 `/api/engineer/tasks` 请求
- 状态应该是 200
- 点击查看Response内容

```bash
curl http://localhost:8800/api/engineer/tasks | python3 -m json.tool | head -30
```

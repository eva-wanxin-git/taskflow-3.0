## 🎯 快速开始

### 1. 解压文件
```bash
tar -xzf 任务所3.0-Blanc-Luxury-Edition-20251128.tar.gz
cd taskflow-v1-2/taskflow-v1-2
```

### 2. 启动服务
```bash
# 启动Dashboard（8831端口）
cd dashboard-test-8831
python3 -m http.server 8831

# 启动API（8800端口）- 可选
cd apps/api
python3 -m uvicorn src.main:app --port 8800
```

### 3. 访问系统
- **Dashboard**: http://localhost:8831
- **API文档**: http://localhost:8800/api/docs

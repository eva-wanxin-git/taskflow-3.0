
```

```

#### Step 4: 追溯父容器
```
在Elements面板中：
1. 一层层往上看父元素
2. 找到第一个有 background: white 的容器

```

## 🎯 可能的原因

### 1. 浏览器缓存（最可能）
- **症状**：代码已修复，但浏览器显示旧版本
- **解决**：强制刷新 Ctrl + Shift + R

### 2. 还有其他容器的高度问题
- **症状**：某个容器设置了 `min-height: 100vh` 或固定高度
- **解决**：需要用开发者工具定位具体容器

### 3. Tab面板的高度继承
- **症状**：Tab内容区域高度过高
- **解决**：检查 `.tab-pane` 的CSS

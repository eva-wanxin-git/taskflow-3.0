## 🚀 部署方法

### 方法1：本地部署（开发测试）

```bash
# 解压
tar -xzf Dashboard-v1.9-20251121.tar.gz

# 进入目录
cd dashboard-v1.9-20251121

# 启动服务器
python3 -m http.server 8820

# 打开浏览器
open http://localhost:8820/
```

### 方法2：换电脑部署

**在旧电脑**：
```bash
# 压缩包已生成
Dashboard-v1.9-20251121.tar.gz (5.7MB)
```

**传输方式**：
- 网盘上传下载
- U盘复制
- 局域网传输
- AirDrop（Mac）

**在新电脑**：
```bash
# 解压
tar -xzf Dashboard-v1.9-20251121.tar.gz

# 进入并启动
cd dashboard-v1.9-20251121
python3 -m http.server 8820

# 打开Cursor，复制提示词
cat 🎯复制给新Cursor-第一句话.txt
# 粘贴到Cursor发送
```

### 方法3：生产服务器部署（Nginx）

```nginx
server {
    listen 80;
    server_name dashboard.yourdomain.com;
    root /var/www/dashboard-v1.9;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

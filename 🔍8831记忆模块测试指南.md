## 💡 如果发现问题

### 搜索框不工作
- F12打开控制台
- 查看是否有JavaScript错误
- 确认searchMemories函数存在

### 创建按钮点击无反应
- 检查控制台错误
- 确认showCreateMemoryDialog函数存在

### 对话框无法关闭
- 点击对话框外的灰色背景
- 或者填写后点击"创建"

### API数据不显示
- 8000端口API未启动是正常的
- 会使用静态数据
- 控制台显示：`[Memory] API not available, using static data`
